# Antibiotic-Resistance-Gene-Detection-in-Klebsiella-pneumoniae
Antibiotic resistance gene detection pipeline using a real multidrug-resistant Klebsiella pneumoniae isolate (SRR24819222). Includes FASTQ QC, trimming, assembly, and prediction of AMR genes using the CARD RGI tool. A lightweight, reproducible workflow for microbial genomics and antimicrobial resistance analysis.
